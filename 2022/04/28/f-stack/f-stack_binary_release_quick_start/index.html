F-Stack快速入门

发表于2022-04-28|更新于2022-04-28|dpdkf-stack
|字数总计:615|阅读时长:2分钟|阅读量:

如何使用

在开始使用 f-stack 二进制发布包之前，请确保您：

  • 连上网络
  • 至少有一张网卡
  • 有root权限
  • 符合dpdk的所有要求
  • dev_name是你网卡的名称，你可以使用ifconfig或者ip addr命令来查看。
  • app_name是你要运行的应用的名称，我们目前提供了nigix和redis。如果你没有提供参数，它将会执行一个helloworld的示例应用，你可以使用该示例来测试环境是否正确配置。
  • 所有的脚本都需要root权限。

支持的linux发行版

f-stack-binary-release 包已经支持了几个经常使用的 linux 版本，你可以直接在这些版本上使用 f-stack 应用程序。需要注意的是，f-stack使用linux内核模块，所以如果你的机器有不同的内核版本，你不能使用这个f-stack-quick-start软件包，需要自己编译f-stack。以下是此软件包现在支持的 linux 版本列表：

Linux Release Kernel
CentOS 7.0 3.10.0-123.el7.x86_64
CentOS 7.2 3.10.0-327.el7.x86_64
CentOS 7.3 3.10.0-514.el7.x86_64
CentOS 7.4 3.10.0-693.el7.x86_64
CentOS 7.5 3.10.0-862.el7.x86_64
RHEL 7.2 3.10.0-327.el7.x86_64
RHEL 7.3 3.10.0-514.el7.x86_64
RHEL 7.4 3.10.0-693.el7.x86_64
RHEL 7.5 3.10.0-862.el7.x86_64
Ubuntu 14.04.5 4.4.0-31-generic
Ubuntu 16.04.4 4.13.0-36-generic
Ubuntu 18.04 4.15.0-20-generic

此外，我们支持腾讯云，如果您在 cloud.tencent.com 中使用云虚拟机，您也可以使用此f-stack-quick-start软件包。
