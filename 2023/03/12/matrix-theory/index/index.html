秩公式

秩定理: r(AB)≤r(A)r(AB) \le r(A)r(AB)≤r(A), 且 r(AB)≤r(B)r(AB) \le r(B)r(AB)≤r(B), 其中AAA为 mmm x nnn 阶矩阵，BBB 为 nnn x ppp 阶矩阵。

备注公式: 若 PPP, QQQ 都可逆，则 r(PA)=r(AQ)=r(A)r(PA) = r(AQ) = r(A)r(PA)=r(AQ)=r(A); 且当 k≠0k \neq 0k=0 时, r(kA)=r(A)r(kA) = r(A)r(kA)=r(A), 特别 r(−A)=r(A)r(-A) = r(A)r(−A)=r(A)。

秩公式: r(ATA)=r(A)=r(AAT)r(A^TA) = r(A) = r(AA^T)r(ATA)=r(A)=r(AAT), AAA为实矩阵。
分块公式: r(A,B)≤r(A)+r(B)r(A, B) \le r(A) + r(B)r(A,B)≤r(A)+r(B), 即 r(A∣B)≤r(A)+r(B)r(A|B) \le r(A) + r(B)r(A∣B)≤r(A)+r(B)

积零公式: 设 AB=0AB=0AB=0 则 r(A)+r(B)≤nr(A)+r(B) \le nr(A)+r(B)≤n, 其中 AAA 为 mmm x nnn 阶矩阵，BBB 为 nnn x ppp 阶矩阵。即：若 AB=0AB = 0AB=0 则 r(A)+r(B)≤nr(A)+r(B) \le nr(A)+r(B)≤n

证: 设 AAA 为 mmm x nnn 阶矩阵， BBB 为 nnn x ppp 阶矩阵，将 BBB 按列分块: B=(β1,...,βp)B = (β_1,...,β_p)B=(β1​,...,βp​)，则 AB=(Aβ1,...,Aβp)=0AB = (Aβ_1, ..., Aβ_p) = 0AB=(Aβ1​,...,Aβp​)=0。
可知 Aβ1=Aβ2=...=Aβp=0Aβ_1 = Aβ_2 = ... = Aβ_p = 0Aβ1​=Aβ2​=...=Aβp​=0, 即 β1,..βpβ_1,..β_pβ1​,..βp​ 都在空间 W={X∣AX=0}W = \{X | AX = 0\}W={X∣AX=0}中，故秩 r(B)=r{β1,...,βp}≤dim(W)=n−r(A)r(B)=r\{β_1,...,β_p\} \le dim(W)=n-r(A)r(B)=r{β1​,...,βp​}≤dim(W)=n−r(A)

补充公式: r(A±B)≤r(A)+r(B)r(A \pm B) \le r(A) + r(B)r(A±B)≤r(A)+r(B), A,BA,BA,B都是 mmm x nnn 阶矩阵

补充定理1: 若 r(A∣B)=r(B)r(A|B)=r(B)r(A∣B)=r(B), 则矩阵方程 AX=BAX=BAX=B必有解X=DX=DX=D, 使得AD=BAD=BAD=B。
引理1: 若 nnn 阶方阵 AAA 满足: r(Ak)=(Ak+1)r(A^k)=(A^{k+1})r(Ak)=(Ak+1), 则存在DDD使得Ak=Ak+1DA^k = A^{k+1}DAk=Ak+1D。
